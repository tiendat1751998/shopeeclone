# CODE STYLE ENFORCEMENT

All generated code MUST:
- resemble senior engineer code
- be human-readable
- be maintainable
- be production-grade

NEVER generate:
- decompiled-looking code
- compiler-style code
- IDE fallback syntax
- autogenerated-looking structures
- unreadable generics
- unnecessary fully-qualified names

Prioritize:
- readability
- maintainability
- explicit imports
- clean structure
- production conventions
