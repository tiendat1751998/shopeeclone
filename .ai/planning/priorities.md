# Engineering Priorities Matrix

## 🔴 Priority 0 (Critical Path - Must Have for Launch)
- User sign-up, JWT validation, roles checking.
- Cart additions, SKU variation selections, checkout validation.
- Inventory deductions and double-entry transaction ledgers.
- Payment integration with VNPay/MoMo gateways.

## 🟡 Priority 1 (System Optimization - Highly Needed)
- Lua script integration for campaign flash sales.
- Search indexing using Elasticsearch vi-analyzers.
- Outbox pattern for state modifications.
- Prometheus dashboard and central Loki monitoring.

## 🔵 Priority 2 (Advanced Additions - Nice to Have)
- Graph recommendation widgets.
- Real-time notification services using WebSockets.
- Coupon combination logic matrix engines.
